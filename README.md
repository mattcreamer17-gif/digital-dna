# DigitalDNA Labs Landing Page

A single-page, bespoke landing page built for Jason Inasi and the DigitalDNA Labs team ahead of eMerge Americas 2026 (Booth 329). The page uses DigitalDNA Labs' own brand tokens (black canvas, navy surface, orange #ff8100 accent, Archivo headings, Inter body) and carries the fit-analysis pitch body in their brand voice, with the Interpretable Context Methodology paper, the Ethics Engine bio, and the Correlation One / Feeld precedent surfaced understated rather than loud. The single CTA routes to Matt Creamer's Calendly. Eduba branding is confined to the footer per sprint rules.
